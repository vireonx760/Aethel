# AethelGUI Roadmap

## 0.3.0 - Developer Preview

AethelGUI 0.3.0 turns the engine-like foundation into a usable developer preview.

### Goals

- Provide `aethel_gui::prelude`.
- Keep the 0.2 retained renderer and scheduler stable.
- Add immediate-style `Ui` construction over retained widget state.
- Add `Response` for widget interactions.
- Add stable widget keys and retained state reuse.
- Stabilize basic layout helpers: row, column, panel, separator, progress bar.
- Add baseline dark theme exports.
- Add examples for basic widgets and retained immediate-style usage.
- Document public, experimental, and low-level API boundaries.

### Non-goals

- WASM support.
- C/C++ wrappers.
- Full accessibility.
- Docking system.
- Multi-window support.
- Full flexbox or grid layout.
- Rich text editing.
- Full animation framework.
